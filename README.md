
 1. Análisis de rendimiento de tiendas

Este análisis compara cuatro tiendas en función de:

- **Ingresos totales (Precio)**
- **Calificación promedio de clientes**
- **Costo de envío promedio**
- **Categorías de productos más y menos vendidas**

### Conclusiones 

- **Tienda 2** es la más rentable, con buen balance entre ingresos, calificaciones y costos.  
- **Tienda 4** presenta bajo volumen de ventas  y alto costo de envío →  
- **Tienda 1 y 3** mantienen rendimiento estable, aunque inferior a la Tienda 2.
